写在前面：避坑

请从 python.org 官方网站 下载并安装。 不要使用带有捆绑或“下载加速器”的第三方入口；并避免安装会修改系统环境的安全/管家类软件（如 360），以减少环境变量被篡改的风险。:contentReference[oaicite:1]{index=1}

1. 检查是否已装 Python

按 Win + R → 输入 cmd 回车，在终端输入：

python --version

若显示版本号，说明已装；没有则继续下一步。这个命令安装后也会再用来验证。:contentReference[oaicite:2]{index=2}

2. 下载安装程序

打开 Downloads，选择 Windows 版本（多数电脑为 64-bit）。 文档中建议安装一个口碑较好的稳定分支（例如 3.11.5）以减少兼容问题；也可以按需选择其他版本。:contentReference[oaicite:3]{index=3}

3. 安装：勾选 PATH，走快速安装

运行安装器，保持默认选项，**选择“Install Now/立即安装”**，并确保勾选：

  • Add Python to PATH（将 Python 加入环境变量）
  • Install launcher for all users（可选）

安装完成后直接关闭即可，**不要点击**“Disable path length limit”。随后重启电脑。:contentReference[oaicite:4]{index=4}

4. 验证安装

重启后再次打开终端：

python --version

看到版本号（例如 Python 3.11.5）即安装成功。:contentReference[oaicite:5]{index=5}

附：常见问题

  • 终端提示不是内部或外部命令：大多是没有勾选 PATH，重新安装并勾选，或手动把安装目录的 python.exe 加入系统 PATH。
  • 执行 python 打开的是微软商店：在“应用执行别名”里关闭 Python 的别名，或直接用 py 启动器。
